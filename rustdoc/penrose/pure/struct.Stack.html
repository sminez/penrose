Struct penrose::pure::Stack

pub struct Stack<T> { /* private fields */ }
Expand description

A Stack can be thought of as a LinkedList with a hole punched in it to mark a single element that currently holds focus. By convention, the main element is the first element in the stack (regardless of focus). Focusing operations do not reorder the elements of the stack or the resulting Vec that can be obtained from calling Stack::flatten.

This is a zipper over a LinkedList. Many of the methods that mutate the structure of the Stack return back a mutable reference so that they are able to be chained.
