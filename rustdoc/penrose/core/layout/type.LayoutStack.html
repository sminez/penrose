Type Definition penrose::core::layout::LayoutStack

source · 
pub type LayoutStack = Stack<Box<dyn Layout>>;
Expand description

A stack of Layout options for use on a particular crate::pure::Workspace.

The Stack itself acts as a Layout, deferring all operations to the currently focused Layout.
