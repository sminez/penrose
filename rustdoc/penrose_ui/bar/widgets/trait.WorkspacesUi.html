fn colors_for_workspace( &self, workspace_meta: &WsMeta, focus_state: FocusState, screen_has_focus: bool, ) -> (Color, Color)

The foreground and background color to be used for rendering a given workspace.

The FocusState provided indicates the current state of the workspace itself, while screen_has_focus is used to indicate whether or not the screen the parent WorkspacesWidget is on is currently focused or not.

fn update_from_state<X>( &mut self, workspace_meta: &[WsMeta], focused_tags: &[String], state: &State<X>, x: &X, ) -> bool
where X: XConn,

Update the UI properties of the parent WorkspacesWidget as part of startup and refresh hooks.

The boolean return of this method is used to indicate to the parent widget that a redraw is now required. If state has not changed since the last time this method was called then you should return false to reduce unnecessary rendering.
